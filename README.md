# Gagan Kalra Career Guidance Website

A production-focused Next.js website for career guidance, mentoring, student success stories, gallery content, and contact enquiries.

## Tech Stack

- Next.js App Router
- React
- TypeScript
- Tailwind CSS
- Framer Motion

## Local Development

```bash
npm install
npm run dev
```

The local site runs at `http://localhost:3000`.

## Deployment Check

```bash
npm run lint
npm run build
```

Deploy the repository to any Next.js-compatible host such as Vercel, Netlify, or a Node server using `npm run build` followed by `npm run start`.
